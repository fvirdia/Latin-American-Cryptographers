The larger Latin American community

This is an example for generating a map of contacts from a #bio-style description. The map was generated automatically from this file.

Blue markers are current work positions, green markers are places of origin. Hover over the marker to see the bio.

The tiles are taken from Mapbox, the geolocation data from OpenStreetMap. It is also possible to use OpenStreetMap tiles, in case Mapbox has bad T&Cs.
See the map with: Mapbox tiles or OpenStreetMap tiles.
